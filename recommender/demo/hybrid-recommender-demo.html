The blend

Serving weights below are the defaults; run_eval re-tunes them by grid search on a validation window each training run. Cold-start users skip CF and content entirely.

score(u, i)  =  0.50 · CF + 0.20 · Content + 0.10 · Popularity + 0.20 · Context

Evaluation protocol

train · day 0–70
val
test

Strict temporal split — weights tuned on days 70–80, reported on days 80–90. Ground truth = click-or-stronger events on items the user never saw in training. No leakage.

Run it yourself

# from the splunk_hec repo root
python -m recommender.run_eval          # train all 4 models, print comparison
python -m pytest tests/test_recommender.py -q
uvicorn recommender.service:app --port 8100
curl -X POST localhost:8100/recommend \
  -H "Content-Type: application/json" \
  -d '{"user_id": 42, "device": "mobile", "hour_bucket": "evening", "k": 5}'
